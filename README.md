## Technologies and requirements
- NestJS
- JavaScript
- TypeScript
- Prisma
- MySQL

## Project
This is a repository that contains an simple NestJS API about Movies developed at Blue EdTech.

## Running the app
Install all dependencies:
```
npm install
```

Then run:

```
npm run start:dev
```

Lastly access [http://localhost:3000](http://localhost:3000).
